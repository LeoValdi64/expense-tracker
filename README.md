# ExpenseWise - Expense Tracker

A professional expense tracking application built with Next.js, TypeScript, and Tailwind CSS v4.

## Features

- **Dashboard** with summary cards: Total Balance, Total Income, Total Expenses, Transaction Count
- **Add Transaction** form with income/expense toggle, categories, and date picker
- **Transaction List** with search, type filter, and category filter
- **Category Breakdown** with visual CSS bar charts
- **Monthly Summary** with comparative income vs. expense view
- **localStorage** persistence â€” data survives page refreshes
- **Dark Theme** with emerald green (income) and red (expenses) accents
- **Mobile-first** responsive design
- **SEO** optimized with Open Graph and Twitter card meta tags

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide React (icons)

## Categories

**Income:** Salary, Freelance, Investment, Gift, Other  
**Expense:** Food, Transport, Housing, Entertainment, Shopping, Health, Education, Bills, Other

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
